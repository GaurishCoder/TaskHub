**TaskHub** is a task management system that allows users to securely manage their daily tasks. The application supports user authentication and ensures that each user can only access and manage their own tasks.

Key functionalities include:

* User registration and login
* JWT-based authentication
* Create, read, update, and delete (CRUD) tasks
* User-specific task access

This project is built with a focus on clean architecture, type safety, and real-world backend practices.
